Data types

Data types specify the type of the data that we enter and the storage capacity of that data type. C language has some predefined set of data types to handle various kinds of data that we can use in our program. These datatypes have different storage capacities.

There are 2 different type of data types in C

  1. Primary data types: - These are fundamental data types in C namely integer(int), floating point(float), character(char) and void.
  2. Derived data types: - Derived data types are nothing but primary datatypes but a little twisted or grouped together like arrays, stuctures, union and pointer.

Data type determines the type of data a variable will hold. If a variable x is declared as int. it means x can hold integer values only. Every variable which is used in the program must be declared as what data-type it is.

Integer type

Integers are used to store whole numbers, Size and range of Integer type on 16-bit machine:

Type Size (Bytes) Range
Int or signed int 2 -32,768 to 32767
Short int or signed short int 1 -128 to 127
Unsigned short int 1 0 to 255
Unsigned int 2 0 to 65535
Long int or signed long int 4 -2,147,483,648 to 2,147,483,647
Unsigned long int 4 0 to 4,294,967,295

Floating point type

Floating types are used to store real numbers.

Type Size (Bytes) Range
Float 4 3.4E-38 to 3.4E+38
Double 8 1.7E-308 to 1.7E+308
Long double 10 3.4E-4932 to 1.1E+4932

Character type

Character types are used to store characters value.

Type Size (Bytes) Range
Unsigned char 1 0 to 255
Char or signed char 1 -128 to 127

Void type

Void type means no value. This is usually used to specify the type of functions which returns nothing.
