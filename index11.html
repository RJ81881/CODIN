Arrays

An array is defined as finite collection of homogenous data, stored in contiguous memory locations.

Examples where Arrays are used

  1. to store Id's of students,
  2. to store list of Employee or Student names,
  3. to store list of numbers or characters

Since arrays provide an easy way to represent data, it is classified amongst the data structures in C. Other data structures in c are structure, lists, queues, trees etc. Array can be used to represent not only simple list of data but also table of data in two or three dimensions.

Declaring an Array

Like any other variable, arrays must be declared before they are used.

For Example : data-type variable_name[size];
int a[10];

Here int is the data type, a is the name of the array and 10 is the size of array. It means array 'a' can only contain 10 elements of int type.Index of an array starts from 0 to size-1 i.e first element of arr array will be stored at a[0] address and the last element will occupy a[9].

Initialization of an Array

After an array is declared it must be initialized. Otherwise, it will contain garbage value(any random value). An array can be initialized at either compile time or at runtime.

Compile time Array initialization

data-type array_name[size] = { list of values };
int score[4]={ 67, 87, 56, 77 };
float area[5]={ 23.4, 6.8, 5.5 };
int marks[4]={ 67, 87, 56, 77, 59 }; // Compile time error

#include<stdio.h>
void main( )
{
int i;
int arr[4]={1,2,3,4};
for(i=0;i<4;i++) {
printf("%d\t",arr[i]);
}
}

Outout :

1 2 3 4

Runtime Array initialization

#include<stdio.h>
void main( )
{
int i;
int arr[4];
printf("Enter Array Elements");
for(i=0;i<4;i++)
{
scanf("%d",&arr[i]);
}
for(i=0;i<4;i++) {
printf("%d\t",arr[i]);
}
}

Two dimensional Arrays

C language supports multidimensional arrays also. The simplest form of a multidimensional array is the two-dimensional array. Both the row's and column's index begins from 0.

Two-dimensional arrays are declared as follows :

data-type array-name[row-size][column-size];
int a[3][4];

An array can also be declared and initialized together. For example :

int arr[][3] = { {0,0,0},
{1,1,1}
};

We have not assigned any row value to our array in the above example. It means we can initialize any number of rows. But, we must always specify number of columns, else it will give a compile time error. Here, a 2*3 multi-dimensional matrix is created.

Runtime initialization of a two dimensional Array

#include<stdio.h>
void main()
{
int arr[3][4];
int i, j;
printf("Enter array element");
for(i = 0; i < 3; i++)
{
for(j = 0; j < 4; j++)
{
scanf("%d", &arr[i][j]);
}
}
for(i = 0; i < 3; i++)
{
for(j = 0; j < 4; j++)
{
printf("%d", arr[i][j]);
}
}
}
