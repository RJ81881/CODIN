Decision making in C

Decision making is about deciding the order of execution of statements based on certain conditions or repeat a group of statements until certain specified conditions are met.

Decision making with if statement

The if statement may be implemented in different forms and The different forms are,

  1. if statement
  2. if else statement
  3. Nested if else statemen
  4. else if statement

if statement

The general form of a simple if statement is :

if(expression)
{
statement 1;
}
statement 2;

If the expression returns true, then the statement 1 will be executed, otherwise statement 1 is skipped and only the statement 2 is executed.

#include<stdio.h>
void main( )
{
int x;
x = 99
if (x>0)
{
printf("x is greater than zero");
}
}

Outout :

x is greater than zero

if else statement

The general form of a if else statement is :

if(expression)
{
statement 1;
}
else
{
statement 2;
}

If the expression returns true, then the statement 1 will be executed, else statement 2 is executed.

#include<stdio.h>
void main( )
{
int x;
x = 99;
if (x>0)
{
printf("x is greater than zero");
}
else
{
printf("x is not greater than zero");
}
}

Outout :

x is greater than zero

Nested if else statement

The general form of a Nested if else statement is :

if( expression )
{
if( expression1 )
{
statement 1;
}
else
{
statement 2;
}
}
else
{
statement 3;
}

if expression is false then statement 3 will be executed, otherwise the execution continues and enters inside the first if to perform the check for the next if block, where if expression 1 is true the statement 1 is executed otherwise statement 2 is executed.

#include<stdio.h>
void main( )
{
int a, b, c;
printf("Enter 3 numbers...");
scanf("%d%d%d",&a, &b, &c);
if(a > b)
{
if(a > c)
{
printf("a is the greatest");
}
else
{
printf("c is the greatest"); }
}
else
{
if(b > c)
{
printf("b is the greatest");
}
else
{
printf("c is the greatest");
}
}
}

Points to Remember

  1. Other than 0(zero), all other values are considered as true.means if(-10) or if(10) is always true.
  2. == must be used for comparison in the expression of if condition, if you use = the expression will always return true, because it performs assignment not comparison.
