Variables

When we want to store any data on our computer, we store it in the computer's memory space. Instead of remembering the complex address of that memory space where we have stored our data, our operating system provides us with an option to create folders and name them, so that it becomes easier for us to find it and access it.

Similarly, in C , when we want to store some data in our program, we can store it in a memory space and name the memory space so that it becomes easier for us to access it.

The naming of an address is known as variable.and Variable is the name of memory location.

Datatype of Variable

A variable in C language must be given a type, which defines what type of data the variable will hold.

it can be an integer or floating number or be a character etc

Rules to name a Variable

  1. Keywords are not allowed as variable name.
  2. Upper and lower case names are treated as different,as C is case-sensitive
  3. Variable name must not start with a digit.
  4. Variable name can consist of alphabets, digits and special symbols like underscore (_) .
  5. Blank or spaces are not allowed in variable name.

Declaration of variables

Declaration of variables should be done before they are used in the program. Declaration does the following things :

  1. It tells the compiler what the variable name is.
  2. It specifies what type of data that the variable will hold.
  3. Declaration is more like informing the compiler that there exist a variable with following datatype which is used in the program.
  4. Until the variable is defined the compiler doesn't have to worry about allocating memory space to the variable.

Defining a variable

Defining a variable means the compiler has to now assign a storage to the variable because it will be used in the program. We can even define multiple variables of same datatype in a single line by using comma to separate them.

Initializing a variable

Initializing a variable means to provide it with a value. A variable can be initialized and defined in a single statement, like: int v =99;
